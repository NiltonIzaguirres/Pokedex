# :boom: Como Executar:
Necessário ter NodeJs, Yarn e Expo instalado na máquina.
* Web
```
# Clone do GitHub
$ git clone https://github.com/NiltonIzaguirres/Pokedex.git
# Entre na pasta
$ cd web
# Instale as dependências
$ yarn install
# Iniciar
$ yarn start 
```
* Mobile
